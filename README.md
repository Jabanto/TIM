# TIM
Tool Inventory Management System con JAVA Fx y Sprint Boot y una base de datos SQLite para la gestión de herramientas.
En este proyecto, utilizaremos la tecnología de interfaz de usuario JavaFX, que se implementará en el framework de Spring Boot. También se agregará una base de datos local, es decir, portable, como H2 o SQLite. Se utilizan estructuras de proyecto Maven y bibliotecas como Hibernate para la persistencia. Además, se agregarán algunas capturas y se creará un espacio de código (code space) para iniciar la aplicación en GitHub con la versión de Java 8.

# Aplicación de Gestión de Herramientas

## Descripción
Esta es una aplicación de escritorio desarrollada con **JavaFX** (FXML) utilizando **Spring boot**. El proyecto implementa el patrón **MVC** (Model-View-Controller) y utiliza **Hibernate** para la gestión de datos con **SQLite** como base de datos.

La solución contiene dos proyectos:
1. **Main**: Contiene los modelos, repositorios, Interfaz de usuario (UI) desarrollada con FXML y la configuración de la base de datos, así como los servicios para la lógica de negocio.
2. **Test**: Proyecto dedicado a las pruebas de los servicios y repositorios más importantes.

## Características
- Aplicación de escritorio con JavaFx y FXML.
- Gestión de datos utilizando Hibernate y SQLite.
- Sistema modular con separación de la lógica de negocio y la UI con patron MVC, donde al ser una aplicacion.
- Proyecto de pruebas para asegurar la calidad del código.

## Instalación
1. Clona el repositorio:
    ```bash
    git clone https://github.com/Jabanto/TIM.git
    ```
2. Ejecuta el proyecto `Main`.
